if-else statements 
- One of the block will be executed 
- else does not have any condition (it has !condition implicitly)
- if can be used without else but else CANNOT be used without if

if(condition) {
    // block 1
}
else {
    // block 2
}


if...else if...else statements -> Ladder statements

1   -> Monday
2   -> Tuesday
7   -> Sunday

if(num === 1) {
    // Monday
}
else if(num === 2) {
    // Tuesday
}
else if(num === 3) {
    // Wednesday
}
else if(num === 4) {
    // Thursday
}
else if(num === 5) {
    // Friday
}
else if(num === 6) {
    // Saturday
} 
else { // 
    // Sunday
}
